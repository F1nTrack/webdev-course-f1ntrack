Lección 4: Introducción a CSS

¡Bienvenido al mundo del CSS! CSS significa "Cascading Style Sheets" y nos permite dar estilo y hacer que nuestras páginas web se vean hermosas.

¿Qué puede hacer CSS?

  • Cambiar colores de texto y fondo
  • Modificar fuentes y tamaños de texto
  • Agregar márgenes y espaciado
  • Crear bordes y sombras
  • Posicionar elementos en la página

¡Esta caja fue creada con CSS!

Observe el fondo verde, el texto blanco y los bordes redondeados.

Selectores CSS básicos

Los selectores nos ayudan a elegir qué elementos queremos estilizar:

  • body - Selecciona todo el cuerpo de la página
  • h1, h2, h3 - Selecciona títulos
  • p - Selecciona párrafos
  • .clase - Selecciona elementos con una clase específica

Colores en CSS

Podemos usar colores de diferentes maneras:

  • Por nombre: red, blue, green
  • Con códigos hexadecimales: #ff6b6b, #4ecdc4
  • Con RGB: rgb(255, 107, 107)
